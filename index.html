Intro to HTML

What is HTML?

HTML stands for Hyper Text Markup Language and is used to create websites.
It provides the skeletal foundation of a website and tells the browser how to display content.

A Basic HTML File

        
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  </head>
  <body>
  </body>
</html>
All HTML documents start with the declaration <!DOCTYPE html>. This tells the browser the document type and helps render the file.
All the HTML is written in between the <html> and </html> tags.
The content displayed on your webpage is written in between the <body> and </body> tags while the settings and not-displayed HTML goes inside the <head> and </head> tags.
A <meta> element contains the settings for how the browser should display the file. The default values coding editors provide are sufficient.

HTML Syntax

An HTML tag is defined using <>. A pair of tags (opening and closing) is considered an element where the closing tag name is preceded by a forward slash (/). All the content of the element goes in between the opening and closing tags:
<tag>Content in the element</tag>

HTML Headers

Create your first HTML element! On your blank file, inside the body tags, create an h1 header: <h1></h1>
Make it a header about your favorite food. 
          
  <!-- Example -->
  <h1>Condiments for a Hot Dog</h1>
There are different levels of headings, ranging from h1 to h6, h1 having the greatest font size and font weight. Try it out!

The Paragraph Tag

The paragraph tag, <p></p>, is used for displaying content such as this. It has a regular font size and weight.
Under your h1 heading, create write a paragraph that describes your favorite food (e.g: what's in it?). 
          
  <p>
    On a hot dog, you can put sauces such as ketchup, mustard, BBQ sauce, mayo, etc. 
    You can also put various toppings like relish, banana peppers, and onions.
  </p>

Element Attributes

All elements have attributes. They are defined inside the opening tag and provide additional information. Think of them as properties that you can change.
To use one, you state the name of the attribute and associate a value to it: 
          
          <tag attribute=""></tag>
To discover element attributes, you can google it or visit a documentation.

Links (Anchor Tags)

What I did there by linking the documentation. That was pretty cool right? That can be done with an anchor tag: <a href="url">link text</a>
The href attribute defines the destination of the link, which can be external or internal (more on that later). You pass the destination URL as the attribute value. This href attribute is mandatory for any anchor tag or else it won't work.
One very useful optional attribute you can add is target="_blank": <a href="www.google.ca" target="_blank">Google link</a>
By adding this attribute, when the user clicks on the link, the browser will automatically open a new tab instead of using the current tab!
Under your paragraph, add a link that allows the user to learn more about your favorite food

HTML Text Formatting

Paragraphs and paragraphs of text can be pretty dull. However, you can style them! Here are some common formatting tags: 
          
            <!-- Bold -->
            This text is <strong>bolded</strong>

            <!-- Italics -->
            This text is <em>italicized</em>

            <!-- Underline -->
            This text is <u>underlined</u>

            <!-- Strikethrough -->
            This text is <del>deleted</del>
You can always find more through a simple Google or documentation search!
Add some of these formatters to your food description paragraph.
If you're up for it, create some more headers (maybe not h1) and paragraphs. Add some more content to your food site and play with some tags!
